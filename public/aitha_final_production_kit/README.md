# Aitha Final Production Logo Kit

Brand: Aitha  
Pronunciation: Aye-tha  
Tagline: The Modern Front Desk

## Included
- SVG source files
- PNG exports
- Transparent-background versions
- Light, dark, orange, and monochrome versions
- Favicon/app icon sizes
- favicon.ico
- Brand sheet PNG/SVG
- Brand color tokens JSON

## Primary colors
- Navy: #101828
- Teal: #2DD4BF
- Orange: #FF7A30
- Slate: #475467
- Light background: #F8FAFC

## Recommended use
Use `aitha_horizontal_light.svg` for website headers and presentation decks.
Use `aitha_icon_light.svg` or `favicon.ico` for browser tabs, app icons, and social avatars.
Use orange/dark versions for ad creative and high-contrast backgrounds.
